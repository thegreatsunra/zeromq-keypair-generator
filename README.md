# zeromq-keypair-generator

> Public/secret key pair generator for ZeroMQ

## Getting started

```bash
git clone git@github.com:thegreatsunra/zeromq-keypair-generator.git

cd zeromq-keypair-generator

npm install
```

## Usage

```bash
cd zeromq-keypair-generator

node index.js
```
